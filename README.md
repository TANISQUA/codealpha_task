# 🎂 Age Calculator

A simple and responsive **Age Calculator** built using **HTML, CSS, and JavaScript**.  
This project allows users to enter their date of birth and instantly calculate their exact age in **years, months, and days**.

---

## Visit the App by clicking [here](https://tanisqua.github.io/codealpha_task/)

## 🚀 Features

- 📅 User-friendly date picker
- ⚡ Instant age calculation
- 🎨 Clean and responsive UI
- 📱 Mobile-friendly design
- 🧮 Calculates:
  - Years
  - Months
  - Days

---

## 🛠️ Technologies Used

- **HTML5** – Structure of the webpage
- **CSS3** – Styling and responsiveness
- **JavaScript (ES6)** – Age calculation logic
